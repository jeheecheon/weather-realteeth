# 화면 디자인

단일 페이지 애플리케이션이다.

## 레이아웃

![레퍼런스 이미지](./images/apple-weather-desktop.png)

- 레퍼런스 이미지처럼, 좌측에는 토글이 가능한 사이드바, 우측은 메인 영역이다.
- 좌측 사이드바에는 즐겨찾기 목록을 카드형으로 표시한다.
- 우측 메인 영역에는 선택된 날씨 정보를 보여준다.
- UI 배치는 첨부된 레퍼런스 이미지를 거의 흡사하게 가져오되, 색감이나 디테일은 다르게 디자인한다.
- 단, 일부 요소는 레퍼런스를 따르지 않는다. [레퍼런스 미적용 항목](./reference-exceptions.md)을 참고한다.
- 색감이나 디테일은 [디자인시스템 문서](../DESIGN.md)을 따른다.

## 반응형

- 데스크탑일 때에는 사이드바를 좌측에 고정한다(좌측 정렬). 사이드바는 윈도우 가장자리에 완전히 붙지 않고 기본 거터만큼 띄운다. 메인 콘텐츠는 사이드바를 제외한 남는 우측 영역 안에서 중앙 정렬한다.
- 윈도우 너비가 모바일 사이즈일 때에는 사이드바 대신 좌측에서 밀려 들어오는 형태의 Drawer이어야 한다. 이 경우, 토글 버튼은 메인 영역의 좌측 상단에 표시한다.
