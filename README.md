# 날씨앱

## 실행 방법

1. `pnpm install && pnpm dev`
2. [http://localhost:5173](http://localhost:5173) 접속

데모 URL: [https://weather.jeheecheon.com](https://weather.jeheecheon.com)

---

## 기능 설명

- 단일 페이지 웹앱
- 좌측 사이드바에서 즐겨찾기 목록 조회/수정/삭제 가능
- 메인 화면에서 현재 위치 날씨 정보 조회 (시간별/일별)

---

## 기술스택

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Tanstack Query
- Shadcn UI
- react-hot-toast
